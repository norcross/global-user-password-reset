== Description ==
Reset all passwords on a site, excluding the current user. Applies globally, so all user account types are affected.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'global-user-password-reset' folder to the `/wp-content/plugins/` directory or install via the WP admin panel
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the 'Password Reset' menu item under the main 'Users' menu.
